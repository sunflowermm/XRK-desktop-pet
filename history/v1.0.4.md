## v1.0.4

### 亮点
- 检查更新不再影响桌宠：新版本通过独立弹窗展示，桌宠照常运行
- 更新弹窗内展示更新说明文档，可选择「更新」或「取消」；点击更新后可最小化到后台，下载完成后点击「立即重启」即可

### 更新
- **独立更新弹窗**：发现新版本时弹出独立窗口（`update-dialog`），内含版本号、Markdown 更新说明、[更新][取消]；点击更新后显示下载进度，窗口可最小化，桌宠不受影响；下载完成后显示 [立即重启][稍后]
- **手动触发下载**：改为用户点击「更新」后再开始下载（`autoDownload = false`），避免后台自动下载占用网络
- **Gitee / GitCode 更新说明**：发布时上传 `releaseNotes.md`，构建前写入 `update-config.json`；应用在 generic 平台检查更新时若未拿到 releaseNotes，会从 baseUrl 拉取 `releaseNotes.md` 并在弹窗中展示，与 GitHub 体验一致
- **已是最新 / 检查失败**：通过系统通知提示，不再占用主界面

### 说明
- 版本号升级为 1.0.4
- 推荐所有用户升级，以获得更顺畅的更新体验
